# Server Downtime Prediction

This project predicts server downtime duration using machine learning regression models.

## Problem Statement
Unexpected server downtime can impact business operations. This project uses historical server metrics to predict downtime duration.

## Approach
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Feature engineering
- Applied multiple regression models

## Models Used
- Linear Regression
- Decision Tree
- Random Forest
- Gradient Boosting

## Evaluation Metrics
- MAE
- RMSE

## Tools Used
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib

## Key Learnings
- Choosing appropriate regression models
- Model evaluation techniques
- Handling real-world noisy data
